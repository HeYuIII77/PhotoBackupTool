# Photo Backup Tool

This is a simple photo backup tool designed to back up photo files from a source directory to a specified target directory, supporting JPEG and RAW format photos.

## Features

- Supports selecting source and target directories for JPEG and RAW files.
- Supports choosing RAW file formats.
- Supports selecting handling methods when files are duplicated.
- Supports preserving folder structure.
- Provides a progress bar to display backup progress.
- Includes logging functionality.

## Usage

1. Open the program.
2. Click the "Select Source Directory" button to choose the directory containing photos to back up.
3. Click the "Select JPEG Target Directory" button to choose the destination directory for JPEG files.
4. Click the "Select RAW Target Directory" button to choose the destination directory for RAW files.
5. Select the RAW file formats you wish to back up.
6. Choose the handling method for duplicate files.
7. Click the "Start Backup" button to begin the backup process.
8. Monitor the progress bar and log to track backup progress.
