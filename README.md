# 照片备份工具

这是一个简单的照片备份工具，用于将照片文件从源目录备份到指定的目标目录，支持处理
**JPEG** 和各类 **RAW** 格式照片以及各类 **视频** 格式。

## 功能特点

-   支持选择源目录和目标目录（JPEG 和 RAW 文件）
-   支持选择 RAW 文件格式（多品牌、多机型）
-   支持选择 视频 文件格式 (多类型)
-   支持在文件重复时选择处理方式
-   支持保留文件夹结构
-   支持选择执行线程数提高备份速率
-   提供进度条显示整体进度以及单个文件进度
-   提供完整日志记录功能

## 界面说明

主界面包含以下区域：

### 1. 文件夹选择区域

-   源目录选择框与浏览按钮
-   JPEG 目标目录选择框与浏览按钮
-   RAW 目标目录选择框与浏览按钮
-   视频 目标目录选择框与浏览按钮

### 2. 格式选择区域

-   RAW 文件格式选择复选框（支持多种相机格式）
-   视频 文件格式选择复选框（支持多种相机格式）

### 3. 备份设置区域

-   重复文件处理方式下拉框
-   线程数选择框
-   操作日志列表框

### 4. 进度显示区域

-   进度条
-   进度百分比标签
-   **开始备份** 按钮

## 运行环境

-   Windows 7 / 8 / 10 / 11
-   .NET Framework 4.x 或 .NET 6+
-   建议以 **管理员权限** 运行

## 使用方法

1.  打开程序
2.  点击 **"选择源目录"** 按钮选择需要备份的照片目录
3.  点击 **"选择 JPEG 目标目录"** 按钮设置 JPEG 文件存储位置
4.  点击 **"选择 RAW 目标目录"** 按钮设置 RAW 文件存储位置
5.  点击 **"选择 视频 目标目录"** 按钮设置 视频 文件存储位置
6.  选择需要备份的 RAW 文件格式以及视频格式
7.  选择文件重复时的处理方式
7.  选择备份时的线程数
8.  点击 **"开始备份"**
9.  通过进度条和日志查看备份进度

## 线程数选择注意事项
-   单个机械硬盘：1（串行）或 2–3，避免寻道抖动。
-   单个 NVMe/SSD 或多盘并行写入：4–16 视情况提高吞吐。
-   目标为网络存储（NAS/SMB）：一般 4–8，受网络带宽与 NAS 并发能力影响。
-   如果目标多个磁盘/目标路径，适当提高并行度能更明显提升性能；如果都是同一慢盘，过高会反而变慢。

## 注意事项

-   确保目标目录有足够的存储空间
-   备份过程中请勿关闭程序
-   网络路径备份需要稳定的网络连接
-   建议先备份少量文件以验证配置正确性
