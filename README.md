# 照片备份工具

这是一个简单的照片备份工具，用于将照片文件从源目录备份到指定的目标目录，支持处理 JPEG 和 RAW 格式照片。

## 功能特点

- 支持选择源目录和目标目录（JPEG 和 RAW 文件）。
- 支持选择 RAW 文件格式。
- 支持在文件重复时选择处理方式。
- 支持保留文件夹结构。
- 提供进度条显示备份进度。
- 提供日志记录功能。

## 使用方法

1. 打开程序。
2. 点击 "选择源目录" 按钮选择需要备份的照片目录。
3. 点击 "选择 JPEG 目标目录" 按钮选择 JPEG 文件的目标目录。
4. 点击 "选择 RAW 目标目录" 按钮选择 RAW 文件的目标目录。
5. 选择需要备份的 RAW 文件格式。
6. 选择文件重复时的处理方式。
7. 点击 "开始备份" 按钮开始备份过程。
8. 查看进度条和日志了解备份进度。
